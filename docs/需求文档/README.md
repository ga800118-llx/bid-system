# 需求文档

> 更新时间：2026-04-21

## 当前说明

本目录主要保留项目早期需求背景和原始业务来源。

项目已经从“上传文件 + AI 提取”为主的早期阶段，进入“基础平台先行、业务模块后续接入”的阶段，因此阅读需求文档时需要注意区分：

- 早期业务需求：项目上传、PDF 提取、Prompt 配置
- 当前真实开发重点：基础平台能力

## 当前真实开发重点

当前优先级以基础平台为准：

- 部门管理
- 用户管理 V2
- 角色权限管理
- 操作日志
- 系统配置
- 字典管理

当前阶段不继续扩展：

- 项目管理
- 上传
- 项目删除
- 定时任务
- 文件存储
- 消息通知
- API Key
- 请求限流
