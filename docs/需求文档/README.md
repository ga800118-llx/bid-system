# 需求文档

## 1. 项目概述

### 1.1 项目背景

招投标是政府采购和工程建设领域最核心的环节之一，传统方式下，招标文件的审阅和关键信息提取完全依赖人工完成，效率低下且容易出错。本系统旨在通过数字化手段，提升招标文件处理效率，降低信息遗漏风险。

### 1.2 项目目标

**核心目标：** 实现招标需求文档的在线管理 + AI 自动提取关键结构化信息。

- 招标文件上传后，系统自动解析 PDF/Word 中的关键字段
- 项目信息结构化存储，支持搜索、查看、下载
- 多人协作查看，支持角色权限区分（普通用户 / 管理员）

### 1.3 术语说明

| 术语 | 解释 |
|------|------|
| 拦标价 | 招标人对投标价格的最高限价，超出则废标 |
| 下限价 | 投标成本警戒线，低于此价可能存在恶意低价竞争 |
| 投标保证金 | 投标人缴纳的保证履约的押金，一般为固定金额或比例 |
| 履约保证金 | 中标后缴纳的保证合同执行的押金 |
| 价格分 | 评标中价格评分的计算方法（如低价优先法、平均价法等） |
| 主观分 | 需要评委主观判断的打分项（如技术方案、售后服务评分细则） |

## 2. 用户需求

### 2.1 用户角色

| 角色 | 说明 |
|------|------|
| 普通用户（user） | 可查看所有项目列表、项目详情、下载原始文件 |
| 管理员（admin） | 拥有普通用户所有权限 + 上传招标文件 + 删除项目 + 管理用户角色 |

### 2.2 功能需求

#### F1：用户注册

- 用户输入邮箱（用户名）和密码即可注册
- 密码经过 MD5 哈希后存储，不明文保存
- 注册成功后自动登录

#### F2：用户登录

- 用户凭邮箱 + 密码登录
- 登录成功后返回 JWT Token，前端存入 localStorage
- Token 有效期 24 小时

#### F3：项目列表

- 展示所有招标项目，支持分页（默认每页10条）
- 支持按项目名称关键词搜索
- 列表展示：项目名称、项目编号、发标单位、开标时间、拦标价
- 点击项目名称进入详情页

#### F4：项目详情

- 展示招标文件的完整结构化信息
- 支持下载原始 PDF/Word 文件
- 管理员可删除项目

#### F5：招标文件上传（仅管理员）

- 支持拖拽上传 PDF、Word（.pdf / .doc / .docx）
- 文件大小限制：单文件最大 100MB
- 上传后由 AI 自动从文件中提取关键字段，提取完成后存入数据库
- 提取完成后自动跳转至项目详情页

#### F6：AI 信息提取

从招标文件中自动识别并提取以下字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| projectName | 项目名称 | ××市智慧城市一期项目 |
| projectCode | 项目编号 | HW-2024-0501 |
| biddingAgency | 招标代理机构 | ××招标有限公司 |
| clientUnit | 发标单位（采购人） | ××市财政局 |
| bidOpenTime | 开标时间 | 2024-06-15T09:30 |
| complaintDeadline | 质疑截止时间 | 2024-06-10T17:00 |
| ceilingPrice | 拦标价（最高限价，元） | 5000000 |
| floorPrice | 下限价（元） | 3000000 |
| contractPayment | 合同付款方式 | 3-3-4（签约30%、验收30%、质保40%） |
| expertComposition | 评审专家组成 | 5人专家组（3位经济类+2位技术类） |
| priceScoreMethod | 价格分计算方法 | 低价优先法：价格分=最低价/投标价×100 |
| subjectiveScoreDetails | 主观分评审细则 | 实施方案（20分）、售后服务（15分）、整体评价（5分） |
| bidBond | 投标保证金（元） | 80000 |
| performanceBond | 履约保证金 | 合同价的5%（即0.05） |

## 3. 非功能需求

### 3.1 性能需求

- 文件上传响应时间 < 10秒（不含AI提取时间）
- AI 提取时间：一般 5-15 秒（视文件大小和页数）
- 列表查询响应时间 < 1秒

### 3.2 安全需求

- 用户密码 MD5 哈希存储
- 所有 API 需携带有效 JWT Token（登录注册除外）
- 管理员操作（上传/删除）需验证 admin 角色

### 3.3 兼容性

- 浏览器：Chrome / Edge / Firefox 最新版
- 支持文件格式：PDF（.pdf）、Word（.doc / .docx）
- 单文件最大 100MB
