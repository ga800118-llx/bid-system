## 当前基础平台范围

已基本完成：

- 部门管理
- 用户管理 V2
- 角色权限管理
- 操作日志
- 系统配置
- 字典管理
- 文件中心增强第二版
- 站内消息与待办联动第一版
- 待办中心增强第一版
- 个人中心 / 账号安全第一版
- 导入导出基础能力第一版

当前阶段明确不做：

- 定时任务
- 外部消息通道（邮件/短信/微信）
- API Key 管理
- 请求限流
- 项目管理、上传、项目删除的权限平台化改造

后续新增需求，应先判断：

1. 是基础平台收尾
2. 还是业务模块接入

不要继续把所有事情都堆进“基础平台”。
