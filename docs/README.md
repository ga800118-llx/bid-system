# 招投标管理系统 - 项目文档

## 文档目录

`
docs/
├── 需求文档/                    # 甲方需求、需求确认书、原始需求文件
│   └── 01_上传原始文件/         # 原始 PDF/Word 需求文件（只读）
├── 开发文档/                    # 技术方案、接口设计、数据库设计
├── 测试文档/                    # 测试用例、测试报告
└── 部署文档/                   # 安装部署、运维手册、环境配置
`

## 文档管理规范

- **命名格式**：功能模块_日期_版本.md（如 登录模块_20260327_v1.md）
- **Markdown 编写的文档，可导出为 Word 供甲方审阅**
- 重要文档及时更新至 Git 仓库，确保版本可追溯
