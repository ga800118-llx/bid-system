# 用户业务组件

## UserFormModal.vue

用于所有“新增用户 / 编辑用户”入口。

适用页面：

- 用户管理
- 部门管理中的部门内新增 / 编辑用户
- 后续任何需要维护用户基础信息、部门归属、角色授权、账号状态的页面

职责边界：

- 组件负责表单展示、基础交互和字段 disabled 状态
- 页面负责接口调用、权限判断、DataScope、保存后刷新和上下文默认值
- 组件不直接调用后端接口

禁止事项：

- 禁止在业务页面重新复制用户新增 / 编辑表单
- 禁止在业务页面直接散写角色多选、部门选择、状态选择
- 如果字段增加，优先扩展本组件，再同步使用页面
